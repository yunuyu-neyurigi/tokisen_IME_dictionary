# 超ときめき♡変換辞書
超ときめき♡宣伝部についてのIMEの変換辞書を作りたい！  
曲名、とき宣用語、などなど、もっと気軽に変換できれば、宣伝もしやすくなりますね！

## 対象のIMEと、いえば～？
Microsoft IMEもしくはGoogle日本語入力ですよね。

## 内容物と、いえば～？
tokisen_songs_*.txt → 曲名辞書  
tokisen_names_*.txt → 人名・固有名詞辞書  
  
註）ファイル名の*の部分は、
* google → Google日本語入力
* microsoft → MS-IME用  

です。

## 使い方と、いえば～？
1. 右側のReleaseからtokisen_dict_～～.zipをDLする。**（Source～ではないので注意！）**
1. 何かしらの文字入力ソフトを起動する．（例：メモ帳，Word，スタートメニューの検索など）
2. タスクバーの通知領域（右側の方）にある「A」または「あ」を右クリック
3. **Google日本語入力の場合**
「辞書ツール」をクリック→左上の「管理」→「新規辞書にインポート」から，ダウンロードしたファイルをインポートする。  
辞書名はわかりやすいようにしてください（例：とき宣 曲名辞書）
1. **Microsoft IMEの場合**
「単語の登録」→左下の「ユーザー辞書ツール」をクリック→画面上部「ツール」→「テキスト ファイルからの登録」から、ダウンロードしたファイルをインポートする。  
註）新規辞書を作成してからインポートすると管理しやすいかも？


## 編集方針（随時追加）
* 超ver　は読みに含めない  
* ラジオ等の推奨ハッシュタグは、番組略称を読みとして、サジェストのみ。  
* 曲の略称は短縮よみとして扱う。  
* ！や？の全角半角は、直前の文字に揃える。
